Cuando inicio un proyecto

  1. Inicializo con el comando: git init
  2. Configurar el usuario: nombre e email git config user.email 'mi email' git config user.name 'mi nombre'
  3. Configuro el repo remoto: git remote add origin https://github.com/usuario/repo.git
  4. Subo los cambios de este proyecto, que es todos los archivos actuales
    Agrego los archivos al paquete (commit)
    git add .
    Cierro el paquete (commit)
    git commit -m 'mensaje del commit'
    Subo el paquete (commit)
    git push origin master Aqui les puedo pedir el usuario y clave de github

    Ahora podemos abrir github y vemos los cambios
