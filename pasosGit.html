Gui rapida para el uso de Git

Cuando inicio un proyecto

  1. Inicializo con el comando: git init
  2. Configurar el usuario: nombre e email git config user.email 'mi email' git config user.name 'mi nombre'
  3. Configuro el repo remoto: git remote add origin https://github.com/usuario/repo.git
  4. Subo todos los archivos de este proyecto:
    1. Agrego los archivos al paquete (commit)
      git add .
    2. Cierro el paquete (commit)
      git commit -m 'mensaje del commit'
    3. Subo el paquete (commit)
      git push origin master Aqui les puedo pedir el usuario y clave de github
  5. Ahora podemos abrir github y vemos los cambios

Para Bajarme un proyecto

  1. Clono con el comando: git clone https://github.com/usuario/repo.git
  2. Configurar el usuario: nombre e email, por las dudas git config user.email 'mi email' git config user.name 'mi nombre'

    3. Ahora pueden modificar archivo y suben los cambios
